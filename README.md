# Cyber Tech Services Website

## Overview
This is a **Cyber Tech Services** website built using **HTML** and **CSS**. It serves as a professional online presence for a technology services company, showcasing their services, portfolio, and contact information.

The website is fully responsive and designed to provide a clean and modern user experience.

---

## Features
- **Responsive Design:** Works on desktops, tablets, and mobile devices.
- **Modern Layout:** Clean and professional design using HTML and CSS.
- **Services Section:** Highlights the technology services offered.
- **Portfolio Section:** Displays completed projects or case studies.
- **Contact Form:** Allows users to get in touch with the company.
- **Navigation Bar:** Smooth scrolling to different sections of the website.
- **Footer:** Contains social media links and company information.

---

## Technologies Used
- **HTML5** – For website structure and content.
- **CSS3** – For styling, layout, and responsiveness.
- **Font Awesome** – For icons.
- **Google Fonts** – For modern typography.
